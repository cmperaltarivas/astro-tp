## Herramientas Habituales

- Procesadores de texto para cartas, informes y comunicaciones internas.
- Hojas de cálculo para control, seguimiento y organización de datos.
- Correo electrónico y plataformas colaborativas para coordinación de tareas.
- Sistemas básicos de gestión para registrar procesos administrativos.

## Documentos que suelen trabajarse

- Informes administrativos
- Formularios internos
- Registros de asistencia o control
- Planillas de seguimiento
- Comunicaciones formales
