El estudiante debe aprender a incorporar piezas de software y complementos necesarios al sistema de red, utilizando herramientas de configuración y mantenimiento de servicios para intranet y extranet, garantizando la conectividad de los servicios.



---

## Plan de Clases

### Clase 1: Introducción a la Incorporación de Software y Complementos (6 horas)

**Objetivo**:  
Conocer las piezas de software y complementos que se pueden incorporar a un sistema de red para optimizar la conectividad y funcionalidad de los servicios de intranet y extranet.

**Contenido**:

- Piezas de software comunes para intranet y extranet: servidores web, bases de datos, aplicaciones de correo, servicios DNS y FTP.
- Complementos para mejorar la seguridad y el rendimiento: cortafuegos, balanceadores de carga, controladores de ancho de banda.
- Introducción a las herramientas de configuración y mantenimiento: webmin, pfSense, Wireshark, entre otros.

**Actividad**:  
Los estudiantes investigan y analizan piezas de software y complementos que pueden mejorar los servicios de una red existente. Luego, discuten cómo estos pueden ser utilizados para aumentar la funcionalidad y seguridad de la red.

**Evaluación**:  
Preguntas dirigidas sobre las piezas de software y complementos que se pueden incorporar en un sistema de red y su función en la intranet y extranet.

**Materiales**:  
Computadoras con acceso a herramientas de configuración de redes, ejemplos de software de servidores, cortafuegos, y complementos.

---

### Clase 2: Instalación de Software y Complementos en el Sistema de Red (6 horas adicionales)

**Objetivo**:  
Instalar y configurar software y complementos en un sistema de red, utilizando herramientas de administración y mantenimiento.

**Contenido**:

- Procedimientos para instalar software y complementos en sistemas operativos de red (Linux y Windows Server).
- Configuración de herramientas de gestión y mantenimiento de redes: configuración básica de cortafuegos, servidores DNS y sistemas de monitorización.
- Verificación de la conectividad de servicios tras la instalación: pruebas de conexión, rendimiento y seguridad.

**Actividad**:  
Los estudiantes instalan y configuran piezas de software y complementos en un entorno de red simulado, utilizando herramientas de administración de redes. Realizan pruebas para verificar la funcionalidad y conectividad de los servicios.

**Evaluación**:  
Preinforme donde los estudiantes describen el software instalado, los complementos configurados y los resultados de las pruebas de conectividad y rendimiento.

**Materiales**:  
Computadoras con acceso a sistemas operativos de red, software de gestión y mantenimiento de redes (p. ej., webmin, pfSense), guías de instalación.

---

### Clase 3: Configuración Avanzada de Complementos y Solución de Problemas (12 horas)

**Objetivo**:  
Configurar complementos avanzados para mejorar la conectividad y seguridad del sistema de red, resolviendo problemas de funcionamiento en servicios de intranet y extranet.

**Contenido**:

- Configuración avanzada de servicios adicionales: balanceo de carga, VPN, configuración avanzada de cortafuegos.
- Diagnóstico y solución de problemas de conectividad y rendimiento en intranet y extranet: uso de herramientas de monitorización y análisis como Wireshark y NetFlow.
- Mantenimiento y actualización de los complementos y software instalados: cómo garantizar la estabilidad y seguridad a largo plazo.

**Actividad**:  
Los estudiantes configuran servicios avanzados en una red simulada (balanceo de carga, VPN), utilizando complementos específicos. Identifican y solucionan problemas de conectividad mediante herramientas de diagnóstico.

**Evaluación**:  
Informe técnico avanzado que documente la configuración de los complementos instalados, los problemas detectados y las soluciones aplicadas para mejorar el rendimiento y la conectividad de los servicios.

**Materiales**:
Sistemas operativos de red, herramientas avanzadas de diagnóstico y configuración de redes, software de cortafuegos, VPN y balanceadores de carga.

### 🧰 Materiales necesarios

| Material | Cantidad | Uso en la clase |
|---|---|---|
| Computadoras con sistema operativo de red (Linux/Windows Server) | 1 por estudiante | Instalar y configurar software y complementos de red |
| Software de gestión de redes (webmin, pfSense) | 1 licencia/instancia por equipo | Administrar y mantener los servicios instalados |
| Herramienta de análisis de tráfico (Wireshark) | 1 por equipo | Verificar la conectividad tras instalar los complementos |
| Guía de instalación de software de servidores | 1 por estudiante | Seguir procedimientos paso a paso durante la instalación |

:::note[🎟️ Ticket de salida]

1. Explica la diferencia entre instalar un servidor web y un complemento de cortafuegos en un sistema de red. ¿Por qué son necesarios ambos?
2. ¿Cómo verificarías que un complemento recién instalado (por ejemplo, un balanceador de carga) está funcionando correctamente en la red?
3. ¿Qué dificultad encontraste durante la instalación o configuración de software hoy y cómo la superaste o intentarías superarla?

:::

### Evaluaciones Finales para el Criterio 1.3

Al finalizar este criterio, los estudiantes deben entregar un informe técnico que documente el proceso de incorporación de piezas de software y complementos, las configuraciones realizadas, y las pruebas de conectividad. La evaluación se basará en la precisión de la configuración, la correcta utilización de las herramientas de administración y la claridad en la documentación del informe.
