El estudiante debe aprender a calcular un esquema de direccionamiento IP para una red de área local (LAN), aplicando conceptos de máscaras de subred IP y asegurando la conectividad eficiente entre los dispositivos de la red.



---

## Plan de Clases

### Clase 1: Introducción a las Máscaras de Subred y Direccionamiento IP (6 horas)

**Objetivo:**
Conocer las máscaras de subred y su aplicación en el cálculo de un esquema de direccionamiento IP.

**Contenido:**

- Explicación de máscaras de subred y su importancia en el direccionamiento IP.
- Cálculo de rangos de direcciones IP utilizando máscaras de subred.
- Dividir una red LAN en subredes eficientes.

**Actividad:**
Los estudiantes realizarán cálculos básicos de máscaras de subred y establecerán rangos de direcciones IP para una red LAN.

**Evaluación:**
Revisión de los cálculos de subred realizados por los estudiantes, asegurando que comprendan cómo dividir una red LAN en subredes utilizando máscaras de subred.

**Materiales:**

- Computadoras
- Calculadoras
- Guías de direccionamiento IP y máscaras de subred

### Clase 2: Cálculo del Esquema de Direccionamiento IP para una Red LAN (6 horas adicionales)

**Objetivo:**
Calcular un esquema de direccionamiento IP para una red LAN, basado en un escenario práctico y aplicando máscaras de subred.

**Contenido:**

- Asignación de direcciones IP estáticas y dinámicas en los dispositivos de red.
- Cálculo del número de subredes y hosts por subred en una red LAN.
- Pruebas de conectividad para verificar el correcto funcionamiento del esquema de direccionamiento.

**Actividad:**
Los estudiantes calcularán un esquema de direccionamiento IP para una red LAN basada en un escenario dado, asignando direcciones IP a los diferentes dispositivos de la red.

**Evaluación:**
Revisión del esquema de direccionamiento IP creado, verificando que cumpla con los requisitos del escenario propuesto y que las direcciones IP estén correctamente asignadas.

**Materiales:**

- Computadoras
- Software de simulación de redes
- Herramientas de cálculo de direcciones IP

### Clase 3: Verificación y Ajuste del Esquema de Direccionamiento IP (12 horas)

**Objetivo:**
Verificar el funcionamiento del esquema de direccionamiento IP en una red LAN y realizar ajustes para optimizar su rendimiento.

**Contenido:**

- Pruebas de conectividad en la red LAN utilizando el esquema de direccionamiento IP.
- Diagnóstico de problemas de direccionamiento y ajustes de las máscaras de subred.
- Optimización del esquema de direccionamiento IP para mejorar la conectividad de la red.

**Actividad:**
Los estudiantes realizarán pruebas de conectividad en una red LAN utilizando el esquema de direccionamiento IP calculado, ajustando los parámetros según sea necesario para optimizar el rendimiento de la red.

**Evaluación:**
Revisión de las pruebas de conectividad y los ajustes realizados, asegurando que el esquema de direccionamiento IP esté optimizado y sea funcional.

**Materiales:**

- Computadoras
- Herramientas de simulación de redes
- Software de monitoreo de red

### Evaluaciones Finales para el Criterio 2.2

Al finalizar este criterio, los estudiantes deberán entregar un **informe técnico** que documente el cálculo del esquema de direccionamiento IP para una red LAN, incluyendo las pruebas de conectividad realizadas y los ajustes efectuados. La evaluación se basará en la precisión del esquema de direccionamiento y la efectividad de las direcciones asignadas.

### 🧰 Materiales necesarios

| Material | Cantidad | Uso en la clase |
|----------|----------|-----------------|
| Calculadoras o herramientas online de subneteo (subnet calculator) | 1 por estudiante | Calcular subredes, rangos de host y broadcast |
| Computadoras con software de simulación (Packet Tracer/GNS3) | 1 por estudiante | Implementar y probar el esquema de direccionamiento IP |
| Planilla de diseño de esquema IP (tabla de subredes) | 1 por estudiante | Registrar y organizar las subredes, hosts y gateways |
| Escenario de red LAN con requerimientos definidos | 1 por grupo | Base práctica para calcular el esquema de direccionamiento |
