El estudiante debe aprender a realizar cálculos y conversiones entre sistemas numéricos (binario, hexadecimal y decimal) para identificar los tipos de direcciones IP de acuerdo con un esquema de direccionamiento.



---

## Plan de Clases

### Clase 1: Introducción a los Sistemas Numéricos (6 horas)

**Objetivo:**
Familiarizarse con los sistemas numéricos utilizados en redes y su relación con el direccionamiento IP.

**Contenido:**

- Explicación de los sistemas binario, decimal y hexadecimal.
- Ejemplos de conversiones entre sistemas numéricos.
- Aplicación de los sistemas numéricos en el direccionamiento IP.

**Actividad:**
Los estudiantes realizarán ejercicios de conversión de números entre binario, decimal y hexadecimal, aplicados a escenarios de direccionamiento IP.

**Evaluación:**
Revisión de los ejercicios de conversión, asegurando que los estudiantes comprendan el proceso y lo puedan aplicar al direccionamiento IP.

**Materiales:**

- Computadoras
- Calculadoras
- Guías de sistemas numéricos

### Clase 2: Aplicación de los Sistemas Numéricos en el Direccionamiento IP (6 horas adicionales)

**Objetivo:**
Aplicar los sistemas numéricos al esquema de direccionamiento IP para identificar tipos de direcciones IP.

**Contenido:**

- Cálculo de direcciones IP utilizando sistemas binario, decimal y hexadecimal.
- Identificación de los rangos de direcciones IP y su clasificación.
- Utilización de máscaras de subred y su relación con los sistemas numéricos.

**Actividad:**
Los estudiantes realizarán cálculos de direccionamiento IP aplicando las conversiones numéricas, identificando las clases de direcciones IP.

**Evaluación:**
Revisión de los cálculos realizados, asegurando que los estudiantes identifiquen correctamente los tipos de direcciones IP y sus rangos.

**Materiales:**

- Computadoras
- Guías de direccionamiento IP
- Herramientas de simulación

### Clase 3: Solución de Problemas Complejos de Direccionamiento IP (12 horas)

**Objetivo:**
Resolver problemas complejos de direccionamiento IP mediante cálculos y conversiones entre sistemas numéricos.

**Contenido:**

- Resolución de escenarios avanzados de subneteo.
- Análisis de situaciones de direccionamiento en redes LAN utilizando sistemas numéricos.
- Aplicación práctica de los conceptos de conversión y cálculo de direcciones IP.

**Actividad:**
Los estudiantes resolverán problemas complejos de subneteo y direccionamiento IP, aplicando conversiones entre binario, decimal y hexadecimal para establecer las direcciones correctas.

**Evaluación:**
Revisión de las soluciones de los problemas complejos, asegurando que los cálculos sean precisos y que los estudiantes dominen los sistemas numéricos aplicados al direccionamiento IP.

**Materiales:**

- Computadoras
- Calculadoras
- Software de simulación de redes

### Evaluaciones Finales para el Criterio 2.1

Al finalizar este criterio, los estudiantes deberán entregar un **informe técnico** que documente los cálculos y conversiones entre sistemas numéricos, aplicando estos conocimientos al direccionamiento IP de una red LAN. La evaluación se basará en la precisión de los cálculos y la correcta identificación de direcciones IP.
